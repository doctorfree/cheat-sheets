# NAME

docker-kill - Kill one or more running containers

# SYNOPSIS

**docker kill \[OPTIONS\] CONTAINER \[CONTAINER...\]**

# DESCRIPTION

Alias for `docker container kill`.

# OPTIONS

**-h**, **--help**\[=false\] help for kill

**-s**, **--signal**="KILL" Signal to send to the container

# SEE ALSO

**docker(1)**
