# NAME

docker-context-ls - List contexts

# SYNOPSIS

**docker context ls \[OPTIONS\]**

# DESCRIPTION

List contexts

# OPTIONS

**--format**="" Pretty-print contexts using a Go template

**-h**, **--help**\[=false\] help for ls

**-q**, **--quiet**\[=false\] Only show context names

# SEE ALSO

**docker-context(1)**
